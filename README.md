# Quora Clone (CRUD App)

## 🚀 Description
This is a simple Quora-like web app built to practice REST APIs and CRUD operations.

## 🛠️ Tech Stack
- Node.js
- Express.js
- EJS
- CSS

## ✨ Features
- Create posts
- Read posts
- Update posts
- Delete posts

## 📚 What I Learned
- REST API structure
- Routing
- Handling requests (GET, POST, PATCH, DELETE).
